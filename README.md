# SerialBridge
##  概要
本ライブラリは、PC(ROS)/Arduino/Mbed間でシリアル通信を行うためのライブラリです。
デバイス間での数値のやりとりを可能にします。  
本ライブラリはマルチプラットフォームに対応するためにC++で書かれています。
